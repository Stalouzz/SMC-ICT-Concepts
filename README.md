## 📖 Overview

This project is a comprehensive, single-page HTML document that serves as a study guide for understanding institutional trading concepts. It breaks down complex trading principles into digestible explanations with both simple and technical definitions, real chart examples, and actionable trade frameworks. Built with love by Miruka Benard.

### What You'll Learn

- **Market Structure** — Identify bullish/bearish trends, Break of Structure (BOS), and Change of Character (CHoCH)
- **Liquidity Concepts** — Understand Sell-Side Liquidity (SSL), Buy-Side Liquidity (BSL), and why price hunts stops
- **Order Blocks** — Spot valid institutional order blocks and know when to trade them
- **Trade Logic** — A complete 7-step framework for building high-probability trades
- **Risk Management** — Proper invalidation rules, stop-loss placement, and take-profit targeting
- **Psychology** — Understand the game between retail traders and institutions

## 🎯 Features

- **Dual Explanations** — Each concept includes simple language for intuition + technical language for precision
- **Real Chart Examples** — EUR/USD, GBP/USD, and GBP/JPY walkthroughs with ASCII visualizations
- **Complete Trade Setups** — Entry, stop-loss, take-profit, and invalidation levels clearly defined
- **One-Page Cheat Sheet** — Quick reference for all key concepts and checklist rules
- **Practice Guide** — Structured 8-week learning path with daily routines
- **Common Mistakes** — 8 frequent errors and how to avoid them
- **Responsive Design** — Works on desktop, tablet, and mobile devices
